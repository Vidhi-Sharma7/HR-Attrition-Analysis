# HR Attrition Analysis

## Project Overview

This project analyzes employee attrition data to identify the key factors influencing employee turnover. The analysis combines Python, SQL, and Power BI to perform data cleaning, exploratory data analysis, business analysis, and dashboard reporting.

## Tools Used

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* MySQL
* Power BI
* Jupyter Notebook
* GitHub

## Project Workflow

1. Data Cleaning and Preparation using Python
2. Exploratory Data Analysis (EDA)
3. Business Analysis using SQL
4. Interactive Dashboard Development using Power BI
5. Project Documentation and Version Control using GitHub

## Key Business Insights

* Overall Attrition Rate: 16.12%
* Total Employees: 1470
* Employees Left: 237
* Research & Development department shows the highest attrition count.
* Laboratory Technicians and Sales Executives have the highest employee turnover.
* Employees working overtime are more likely to leave the organization.
* Work-Life Balance appears to influence attrition patterns.
